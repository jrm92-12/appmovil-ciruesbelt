Proceso de creacion y ejecucion del proyecto
Para desarrollar la aplicacion movil ciruesbelt-app, se siguieron estos comandos desde la creacion del proyecto hasta su ejecucion en Android:

Crear el proyecto Expo
npx create-expo-app@latest ciruesbelt-app

2. Ingresar a la carpeta del proyecto
cd ciruesbelt-app

3. Instalar las dependencias base
npm install

4. Instalar Firebase
npm install firebase

5. Instalar el selector desplegable
npm install @react-native-picker/picker

6. Instalar el selector de fecha y hora
npx expo install @react-native-community/datetimepicker

7. Ejecutar la aplicacion en Android
npm run android
